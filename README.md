# Implementation-of-lexical-analyzer
lab experiment 1
# Aim
To design and implement a lexical analyzer that identifies tokens from a source program.
# Problem statement
Develop a lexical analyzer that classifies tokens into keywords, identifiers,
operators, separators, and numbers.
# Algorithms
1. Read input source code from a file
2. Split source code into tokens
3. Identify token types using predefined rules
4. Display the tokens with their classifications
